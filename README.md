## ðŸ›  Installation & Set Up

1. Init a new repo:

   ```sh
   git init
   ```

2. Clone the repo:

   ```sh
   git clone https://github.com/sportiz91/next-my-portfolio.git
   ```

3. CD to the folder:

   ```sh
   cd next-my-portfolio
   ```

4. Install all dependencies:

   ```sh
   npm install
   ```

5. Run it locally on your machine:

   ```sh
   npm run dev
   ```
